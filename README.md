# OG Preview - URL Metadata Viewer

A simple tool to preview Open Graph meta tags from any URL. Built for developers who need to preview how their links will look when shared on social media.

## Features
- Paste any URL and see its OG metadata
- Preview title, description, image, and site name
- Mobile and desktop preview
- Copy the previewed image URL

## Tech Stack
- Single HTML file with vanilla JS
- Uses a CORS proxy to fetch metadata
- Deploy anywhere (Netlify, Vercel, Render)

## Deployment
```bash
# Deploy to Vercel
vercel --prod

# Or use the deploy skill
```
